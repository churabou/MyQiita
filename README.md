# MyQiita

- QiitaClientをピンクで作る。
- git-push-hackerthonで後悔したので同じOauth2のサンプルとして作った。


![](https://github.com/churabou/MyQiita/blob/master/demo.gif)

設計は一般的なMVVMと同じ感じ+Navigatorクラスを追加した感じ。

- view
- controller
- viewModel
- navigator

RxSwiftを勉強のため使って見た。
